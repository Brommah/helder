## 1. Wat is de Wkb? {#wat-is-de-wkb}

De **Wet kwaliteitsborging voor het bouwen (Wkb)** is sinds **1 januari 2024** van kracht. Deze wet verandert fundamenteel hoe kwaliteit bij nieuwbouw wordt geborgd.

### De Belangrijkste Veranderingen

| Voorheen | Nu (met Wkb) |
|----------|--------------|
| Gemeente controleert tijdens bouw | Onafhankelijke kwaliteitsborger controleert |
| Beperkte documentatieplicht | Verplicht consumentendossier voor 20 jaar |
| Oplevering op vertrouwen | Gereedmelding met bewijs van kwaliteit |
| Beperkte aansprakelijkheid | Uitgebreide aannemersuaansprakelijkheid |

### Waarom Dit Belangrijk Is

- **Voor bouwers**: Meer administratie, maar ook meer bescherming bij klachten
- **Voor kopers**: Eindelijk zekerheid over wat er in hun huis zit
- **Voor iedereen**: Hogere bouwkwaliteit als norm

## 2. Wanneer is de Wkb van Toepassing? {#wanneer-van-toepassing}

### Wel Wkb-plichtig

- [ ] Nieuwbouw woningen (alle typen)
- [ ] Verbouwingen met vergunning (gevolgklasse 1)
- [ ] Aan- en uitbouwen > 25m²
- [ ] Dakkapellen en dakopbouwen
- [ ] Interne verbouwingen met constructieve wijzigingen

### Niet Wkb-plichtig (voorlopig)

- Bestaande bouw zonder vergunning
- Klein onderhoud
- Gevolgklasse 2 en 3 (tot nader order)

### Twijfel?

Controleer altijd bij uw gemeente of kwaliteitsborger.

## 3. Het Consumentendossier: Complete Checklist {#consumentendossier-checklist}

Dit is de kernverplichting van de Wkb: bij oplevering moet u een compleet consumentendossier aan de koper overhandigen. Dit dossier moet **20 jaar bewaard** blijven.

### A. Tekeningen en Ontwerp

- [ ] **As-built tekeningen** (zoals daadwerkelijk gebouwd)
- [ ] Plattegronden per verdieping
- [ ] Geveltekeningen
- [ ] Doorsneden
- [ ] Situatietekening
- [ ] Constructietekeningen
- [ ] Installatietekeningen (E, W, HVAC)

### B. Berekeningen

- [ ] **Constructieberekening** (gecontroleerd door constructeur)
- [ ] **BENG-berekening** (energieprestatie)
- [ ] Funderingsadvies
- [ ] Daglichtberekening
- [ ] Ventilatieberekening (CO2, balans)
- [ ] Akoestiekberekening (indien van toepassing)

### C. Materiaalspecificaties

- [ ] **Productbladen** van alle gebruikte materialen
- [ ] CE-markeringen
- [ ] Prestatiecertificaten
- [ ] Leveranciersinformatie
- [ ] Kleurcodes en afwerkingen

### D. Installatie-instructies

- [ ] **Handleidingen** voor alle systemen
- [ ] Bedieningsinstructies verwarming
- [ ] Bedieningsinstructies ventilatie
- [ ] Bedieningsinstructies warmtepomp/CV
- [ ] Smart home instructies (indien aanwezig)
- [ ] Zonwering/screens bediening

### E. Onderhoudsvoorschriften

- [ ] **Onderhoudsschema** per materiaal/installatie
- [ ] Schilderonderhoudsplan
- [ ] HVAC onderhoudsschema
- [ ] Dakonderhoud instructies
- [ ] Gevelonderhoud instructies
- [ ] Riolering/HWA onderhoud

### F. Garantiebewijzen

- [ ] **Woningborg garantiecertificaat** (indien van toepassing)
- [ ] Garantie kozijnen/ramen
- [ ] Garantie dakbedekking
- [ ] Garantie installaties
- [ ] Garantie keuken/sanitair
- [ ] Garantie gevelbekleding
- [ ] Garantiecertificaten onderaannemers

### G. Keuringsrapporten

- [ ] **Rapport kwaliteitsborger** (verklaring)
- [ ] Inspectierapporten per fase
- [ ] Druktesten riolering
- [ ] Luchtdichtheidsmeting
- [ ] Thermografische opnames (indien uitgevoerd)
- [ ] Funderingsinspectie

### H. Energiedocumentatie

- [ ] **Energielabel** (definitief)
- [ ] BENG-berekening (as-built)
- [ ] Installatiespecificaties zonnepanelen
- [ ] Warmtepomp specificaties
- [ ] Isolatiewaarden per element

### I. Aanvullende Documenten

- [ ] Bouwvergunning
- [ ] Constructief akkoord
- [ ] Borgingsplan kwaliteitsborger
- [ ] Risicobeoordeling
- [ ] Proces-verbaal oplevering
- [ ] Contactgegevens alle partijen

## 4. De Kwaliteitsborger {#kwaliteitsborger}

### Wat Doet Een Kwaliteitsborger?

De kwaliteitsborger is een onafhankelijke partij die controleert of uw bouwproject voldoet aan alle technische eisen. Dit is **verplicht** voor Wkb-plichtige projecten.

### Wanneer Inschakelen?

| Fase | Actie |
|------|-------|
| **Voor vergunning** | Kwaliteitsborger selecteren en borgingsplan opstellen |
| **Bij vergunning** | Borgingsplan indienen bij gemeente |
| **Tijdens bouw** | Inspecties op kritieke momenten |
| **Bij oplevering** | Verklaring kwaliteitsborger voor gereedmelding |

### Hoe Vind Ik Een Kwaliteitsborger?

Zoek een kwaliteitsborger via het TloK register:
- Controleer of ze geregistreerd zijn
- Vraag referenties
- Vergelijk tarieven (€2.000 - €5.000 afhankelijk van project)

### Let Op

De kwaliteitsborger moet **onafhankelijk** zijn - niet de architect of aannemer van het project.

## 5. Fasering en Timing {#fasering}

### Timeline: Van Vergunning tot Oplevering

```
VERGUNNINGAANVRAAG
└── Borgingsplan opstellen
└── Kwaliteitsborger selecteren
    │
    ▼
START BOUW
└── Bouwmelding indienen (4 weken voor start)
└── Documenten verzamelen vanaf dag 1
    │
    ▼
TIJDENS BOUW
└── Inspecties kwaliteitsborger
└── Foto's maken per fase
└── Documenten verzamelen per fase
    │
    ▼
VOOR OPLEVERING
└── Verklaring kwaliteitsborger aanvragen
└── Consumentendossier samenstellen
└── Gereedmelding voorbereiden
    │
    ▼
OPLEVERING
└── Gereedmelding indienen bij gemeente
└── Consumentendossier overdragen aan koper
└── Archivering voor 20 jaar
```

### Kritieke Deadlines

| Moment | Deadline | Document |
|--------|----------|----------|
| Start bouw | 4 weken voor | Bouwmelding |
| Gereedmelding | 2 weken voor oplevering | Verklaring kwaliteitsborger |
| Archivering | 20 jaar | Volledig consumentendossier |

## 6. Veelgemaakte Fouten {#veelgemaakte-fouten}

### Top 10 Fouten om te Vermijden

1. **Te laat beginnen met documenteren**
   > Start vanaf dag 1, niet pas bij oplevering

2. **Geen fotobewijs per fase**
   > Maak structureel foto's met datum/tijdstempel

3. **Ontbrekende garantiebewijzen onderaannemers**
   > Eis garanties bij elke factuur

4. **Verouderde tekeningen gebruiken**
   > Alleen as-built tekeningen in dossier

5. **Geen borgingsplan bij vergunning**
   > Vergunning wordt geweigerd zonder plan

6. **Kwaliteitsborger te laat inschakelen**
   > Minimum 6 weken voor vergunningaanvraag

7. **Documenten alleen digitaal óf alleen fysiek**
   > Zorg voor beide (backup!)

8. **Onduidelijke archiveringsstructuur**
   > Gebruik vaste mappenstructuur

9. **Geen overzicht van alle partijen**
   > Contactlijst van alle onderaannemers

10. **Bouwmelding vergeten**
    > 4 weken voor start, anders boete!
