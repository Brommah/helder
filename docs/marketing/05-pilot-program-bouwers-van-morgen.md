# Bouwers van Morgen
## Pilot Program Design & Agreements

**Program Name:** Bouwers van Morgen (Builders of Tomorrow)  
**Duration:** 12 weeks  
**Pilot Size:** 5 builders  
**Start Date:** February 2026

---

## Program Overview

### What It Is

"Bouwers van Morgen" is an exclusive pilot program for forward-thinking Dutch builders who want to:
- Simplify Wkb compliance
- Deliver exceptional customer experiences
- Be first-movers in construction documentation technology

### What Participants Get

| Benefit | Value | Details |
|---------|-------|---------|
| **Free Platform Access** | €1,800 | Full access during 12-week pilot |
| **Personal Onboarding** | €500 | 2-hour onboarding call + setup |
| **Priority Support** | €300 | Direct WhatsApp line to team |
| **Feedback Influence** | Priceless | Shape product roadmap |
| **Early Adopter Badge** | Future value | Recognition as founding partner |
| **Extended Free Period** | €600 | 3 extra months free after pilot |
| **Total Value** | **€3,200+** | All free for pilot participants |

### What We Ask In Return

| Commitment | Description |
|------------|-------------|
| **Active Usage** | Document at least 1 project during pilot |
| **Feedback Sessions** | 30-min call every 2 weeks (6 total) |
| **Testimonial** | If satisfied, provide quote/testimonial |
| **Case Study** | Allow us to document your experience |
| **NPS Survey** | Complete survey at end of pilot |

## Ideal Pilot Partner Profile

### Must-Have Criteria

- [ ] Active in nieuwbouw (new construction)
- [ ] Building 10-50 homes per year
- [ ] Located in Utrecht, Noord-Brabant, or Gelderland
- [ ] At least one project starting within pilot period
- [ ] Decision maker available for kickoff call
- [ ] Team uses WhatsApp daily

### Nice-to-Have

- Previous experience with construction software
- Existing pain with Wkb compliance
- Interest in technology/innovation
- Multiple concurrent projects

### Exclusion Criteria

- No active projects during pilot period
- Exclusively renovation (not nieuwbouw)
- Not Wkb-applicable projects only
- Unable to commit to feedback sessions

## Feedback Session Structure

### Bi-Weekly Call Agenda (30 minutes)

1. **Opening (5 min)**
   - How's the project going?
   - Any blockers?

2. **Feature Usage Review (10 min)**
   - What have you used most?
   - What haven't you tried yet?
   - Any confusion?

3. **Pain Points (10 min)**
   - What's frustrating?
   - What's missing?
   - What would make this better?

4. **Feature Request (5 min)**
   - If you could add one thing, what would it be?
   - Prioritize: nice-to-have vs. must-have

### Feedback Tracking

All feedback logged in Notion database:
- Date
- Builder name
- Category (bug, feature, UX, praise)
- Priority (low, medium, high, critical)
- Status (new, reviewed, planned, done, won't do)

## FAQ for Prospective Pilots

### Common Questions

**Q: Is dit echt gratis?**
> Ja, volledig. Geen hidden costs. Wij investeren in feedback, niet in sales.

**Q: Hoeveel tijd kost het?**
> Onboarding: 2 uur. Daarna: 15 minuten per week voor feedback calls. Het platform zelf bespaart tijd.

**Q: Wat gebeurt er na de pilot?**
> Je krijgt 3 maanden gratis. Daarna kun je kiezen voor een betaald abonnement (tegen pilot-korting) of stoppen.

**Q: Zijn mijn data veilig?**
> Ja. Alles is encrypted. We zijn AVG-compliant. Je data blijft van jou.

**Q: Kan ik al mijn projecten toevoegen?**
> Ja, onbeperkt. Hoe meer je gebruikt, hoe waardevoller de feedback.

**Q: Moet ik software installeren?**
> Nee. Woningpaspoort werkt via de browser. Builders gebruiken WhatsApp.
